# BMKG Cuaca Scraper

A simple web scraper to fetch weather data from the BMKG website.

## Requirements
- Python 3.x
-  `requests` library
-  `beautifulsoup4` library
-  `pandas` library 
-  `keyboard` library
-  `prompt_toolkit` library

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python3 bmkg_cuaca_scraper.py
```
